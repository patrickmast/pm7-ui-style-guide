PM7 Input Rules - Complete Integration Guide

Package: pm7-ui-style-guide

Component: PM7 Input Rules (Styling rules for input fields, not a component)

CSS File: No separate CSS file - rules are applied via JavaScript/CSS

Dependencies: None - pure styling rules

Overview

The PM7 Input rules provide styling guidelines for input fields to ensure consistent appearance across all PM7 applications. This module exports configuration rules rather than a complete input component, allowing maximum flexibility with existing input implementations.

Important: This is a rules-based approach, not a component. You apply these rules to your existing input elements.

Installation

npm install pm7-ui-style-guide

# No peer dependencies required - pure styling rules

Import

import { inputRules } from 'pm7-ui-style-guide';

Basic Usage with Existing Components

import { inputRules } from 'pm7-ui-style-guide';

function StyledInput() {
  return (
    <input
      style={{
        border: inputRules.alwaysShowBorder ? '1px solid #D4D4D4' : 'none',
        borderRadius: '6px',
        padding: '0.5rem',
      }}
      className="focus:border-blue-500" // Use #1C86EF for PM7 primary
      placeholder="Enter text..."
    />
  );
}

API Reference

inputRules Object

Property Type Value Description
focusBorderColor string 'primary' Indicates to use PM7 primary color (#1C86EF) for focus border
alwaysShowBorder boolean true Input fields should always have a visible border

Color Values

When focusBorderColor is 'primary', use the PM7 primary color:

  • PM7 Primary: #1C86EF (bright blue)
  • Border Default: #D4D4D4 (light gray)

Dark Mode Support

For dark mode implementations, adjust colors accordingly:

import { inputRules } from 'pm7-ui-style-guide';

function DarkModeInput({ isDark }) {
  const borderColor = isDark ? '#525252' : '#D4D4D4';
  const backgroundColor = isDark ? '#262626' : '#ffffff';
  const textColor = isDark ? '#ffffff' : '#333333';
  
  return (
    <input
      style={{
        border: inputRules.alwaysShowBorder ? `1px solid ${borderColor}` : 'none',
        backgroundColor,
        color: textColor,
        borderRadius: '6px',
        padding: '0.5rem 0.75rem',
        transition: 'border-color 0.2s ease-in-out'
      }}
      className="focus:border-blue-500" // PM7 primary #1C86EF
      placeholder="Enter text..."
    />
  );
}

Integration with Popular Libraries

React Hook Form Integration

import { useForm } from 'react-hook-form';
import { inputRules } from 'pm7-ui-style-guide';

function FormWithValidation() {
  const { register, handleSubmit, formState: { errors } } = useForm();
  
  const getInputClassName = (fieldName) => {
    const hasError = errors[fieldName];
    const borderClass = inputRules.alwaysShowBorder 
      ? `border ${hasError ? 'border-red-500' : 'border-gray-300'}` 
      : 'border-0';
    
    return `${borderClass} rounded-md px-3 py-2 focus:outline-none ${
      hasError ? 'focus:border-red-500' : 'focus:border-blue-500'
    }`;
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input
        {...register('email', { required: true })}
        className={getInputClassName('email')}
        placeholder="Email"
      />
      {errors.email && <span className="text-red-500">Email is required</span>}
    </form>
  );
}

Best Practices

  • Always apply PM7 rules: Use inputRules.alwaysShowBorder to ensure consistent border visibility
  • Use PM7 primary color for focus: Apply #1C86EF for focus states when focusBorderColor is 'primary'
  • Maintain accessibility: Ensure sufficient color contrast for all states
  • Consistent padding: Use 0.5rem 0.75rem for consistent input sizing
  • Smooth transitions: Add transition: 'border-color 0.2s ease-in-out' for better UX
  • Support validation states: Adjust border colors for error and success states
  • Dark mode consideration: Adjust background and text colors for dark themes

Common Usage Patterns

Utility Function for Consistent Styling

import { inputRules } from 'pm7-ui-style-guide';

// Create a utility function for consistent input styling
export const getPM7InputStyle = (options = {}) => {
  const { hasError = false, isSuccess = false, isDark = false } = options;
  
  let borderColor = '#D4D4D4';
  if (hasError) borderColor = '#ef4444';
  else if (isSuccess) borderColor = '#22c55e';
  else if (isDark) borderColor = '#525252';
  
  return {
    border: inputRules.alwaysShowBorder ? `1px solid ${borderColor}` : 'none',
    borderRadius: '6px',
    padding: '0.5rem 0.75rem',
    fontSize: '0.875rem',
    outline: 'none',
    transition: 'border-color 0.2s ease-in-out',
    backgroundColor: isDark ? '#262626' : '#ffffff',
    color: isDark ? '#ffffff' : '#333333'
  };
};

// Usage
function MyInput(props) {
  return (
    <input
      {...props}
      style={getPM7InputStyle({ hasError: props.error, isDark: props.theme === 'dark' })}
    />
  );
}
